## 1. Теоретический блок (ответы на контрольные вопросы)

### A1. Что такое стек вызовов (call stack) в Python и как его увидеть?
**Стек вызовов** - это структура данных, хранящая информацию о всех активных вызовах функций. 
При ошибке Python выводит **traceback**, который читается **снизу-вверх**.
Для просмотра стека можно использовать:
- `traceback.print_exc()` - вывод traceback
- `pdb` - команды `where` или `w`

### A2. В чем разница между except: и except Exception as e:?
- `except:` - перехватывает ВСЕ исключения (включая системные). Антипаттерн!
- `except Exception as e:` - перехватывает только наследуемые от `Exception`. Рекомендуемый способ.

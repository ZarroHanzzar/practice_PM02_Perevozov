# Booking System - Service Layer Implementation

## Вариант 13: Квоты на бронирования (Circuit Breaker)

### Описание
Система управления бронированиями отелей с ограничением количества бронирований в день для одного пользователя с использованием паттерна Circuit Breaker.

### Основные функции
- Управление отелями и номерами (CRUD)
- Бронирование номеров с проверкой доступности
- **Ограничение на количество бронирований в день (3 бронирования)**
- **Circuit Breaker для защиты от превышения квоты**
- Отмена бронирования с возвратом квоты
- Расчет стоимости с учетом сезонности
- Поиск доступных номеров

### Технологии
- Python 3.10+
- Pydantic V2 (валидация)
- Pytest (тестирование)
- In-Memory репозитории

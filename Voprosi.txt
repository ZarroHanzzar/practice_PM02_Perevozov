1. ПМ.02 формирует компетенции в области разработки, отладки и тестирования программных модулей, проектирования баз данных и создания пользовательских интерфейсов.
2. Стек Python + MySQL + tkinter выбран из-за кроссплатформенности, простоты Python, надежности MySQL и удобства tkinter для обучения. Альтернативы: PostgreSQL (вместо MySQL), PyQt или Kivy (вместо tkinter), Java + JDBC + Swing.
3. ER-диаграмма — это модель «сущность-связь». Она нужна для визуализации структуры базы данных: какие данные хранятся и как они связаны между собой.
4. Существуют связи: «один к одному» (страна — столица), «один ко многим» (автор — книги), «многие ко многим» (студенты — курсы). В промежуточной таблице для связи «многие ко многим» создается составной первичный ключ.
5. Git — система контроля версий. **Commit** сохраняет изменения в локальном репозитории, а **push** отправляет эти коммиты на удаленный сервер.
6. В командной строке выполнить: `python --version` и `python -m tkinter` (должно открыться пустое окно).
7. Не оставлять компьютер без присмотра, блокировать экран, использовать сложные пароли, не переходить по подозрительным ссылкам, регулярно обновлять ПО, делать резервные копии данных.
8. Это localhost** — специальный адрес, указывающий на «этот же компьютер». Используется для тестирования сетевых приложений без выхода в интернет.
9. 1. `git add <имя_файла>` или `git add .`
2. `git commit -m "Описание изменений"`
3. `git push`
10. **README.md** содержит описание проекта, инструкции по установке, запуску и использованию. Это первое, что читает пользователь или разработчик, знакомясь с проектом.
